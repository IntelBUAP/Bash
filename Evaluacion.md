#Realize un script que funcione como interprete de ordenes el cual debera cumplir lo siguiente:

- Un comando para realizar busquedas por tamaño, nombre, usuario, permisos, fechas.
- Un comando que nos permite realizar compresion de archivos por medio de .tar.bz, tar.gz, zip, rar.
- Un comando para mostrar archivos con las carpetas a colores.
- Un comando que te permita moverte entre carptetas.
- Un comando que muestre el contenido de un archivo.
Los comandos deberan estar en español y no son validos los alias todo debera estar dentro de un script y ese mismo script debera tener las validaciones necesarias para que no salga de manera repentina.
